# 340B CFO & BizDev Dashboard

Role-based Streamlit dashboard for:
- 340B revenue analytics
- Gout infusion tracking
- BizDev attribution
- Doctor onboarding

## Roles
- Admin: full access
- BizDev: sees only their doctors & revenue
- Investor: aggregated views only

## Run locally
pip install -r requirements.txt
streamlit run app.py
